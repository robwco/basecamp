hey jason! here is a closer look at a few more design elements i worked on inside charm.

in charm, once you've read an email, you use this interface to process it. a lot of the time-saving features of the app come from this screen because you can do multiple things at once. reply, categorize, assign to a team member, and assign to a case, and archive – all with one click. i added a grid to help label each ui element and used apple's SF pro font as a secondary font to style secondary elements.
